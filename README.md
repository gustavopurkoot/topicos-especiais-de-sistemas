# 💻 Tópicos Especiais de Sistemas — 2026

Repositório com os exercícios e projetos desenvolvidos na disciplina **Tópicos Especiais de Sistemas** da faculdade, utilizando **C# com .NET**.

---

## 📚 Conteúdo por Aula

| Aula | Conteúdo |
|------|----------|
| Aula 1 | Variáveis, tipos de dados, entrada/saída, `if/else` |
| Aula 2 | `for`, `while`, tabuada, potência, par/ímpar |
| Aula 3 | Classes, objetos, métodos, construtores |
| Aula 4 | Classes com controle de estoque, métodos de negócio |
| Aula 5 | Arrays de objetos, carrinho de livros com `switch` |
| Aula 6 | `List<T>`, cadastro, busca e remoção de produtos |
| Aula 7 | API REST com ASP.NET Core, rotas GET e POST |
| Lista | Exercícios complementares |
